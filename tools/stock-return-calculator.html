What is a Stock Return Calculator?

A Stock Return Calculator helps investors calculate profit or loss from buying and selling shares.

How Stock Return Works

The calculator compares the purchase price with the selling price and multiplies the difference by the number of shares.

Example

If you buy 100 shares at ₹200 and sell them at ₹250, your total profit would be ₹5,000.

Why This Tool is Useful

This tool helps investors analyze trading performance and investment returns.
